Technicals

There are three components to Carolum:

  • Static webpage - the website you are currently on. It provides information about Carolum, and contains links to everything. The site is styled using Tachyons and a bit of Animate.css.

  • Carolum "server" - In order for this project to be open-source, FireBase hosts the server. FireBase data is retrieved using client-side JavaScript, along with authentication.

  • Carolum clients - This suite interacts with the FireBase "server", containing mobile, desktop, and a web client. Vue.js provides structuring and organization to the JavaScript doing the heavy lifting of authentication, and submitting and retrieving data.
All code can be found on GitHub.
